# Week 1 — 基础概念说明

> 本文件记录 AI × Web3 核心概念的理解，作为后续学习的基础。

## AI 方向

### LLM（大语言模型）
- **是什么**: 基于 Transformer 架构的神经网络，通过海量文本训练，学习语言模式
- **工作方式**: 给定上下文，预测最合理的下一个 token 序列（概率生成）
- **擅长**: 语言理解、代码生成、推理、翻译
- **不擅长**: 精确事实记忆、确定性计算、跨会话状态保持

### Prompt（提示词）
- **是什么**: 输入给 LLM 的文本，引导模型生成特定输出
- **四个控制层面**:
  - 上下文窗口：模型能看到多少信息（"工作内存"）
  - 系统指令：设置身份、语气和边界
  - 提示词：当前任务意图
  - 工具调用：让模型从"说话"变为"做事"

### Workflow（工作流）
- **是什么**: 预定义的任务流程，模型是其中一个节点
- **特点**: 路径固定、可复现、可调试
- **适用**: 流程固定、需要一致性的任务

### Agent（智能体）
- **是什么**: 模型自主规划、动态调用工具、跨轮管理状态
- **核心组件**:
  - 状态管理：多节点共享读写同一 State
  - 长期记忆：跨 session 存储与召回
  - MCP：LLM 与外部工具的统一连接协议
  - Skills：可复用的高层指令集
  - Tool Calling：模型输出结构化请求，框架执行后回传结果
  - Tracing：可视化 agent 执行链
  - Guardrails：输入输出验证规则
  - Handoff：子任务完成后移交控制权
  - 错误恢复：重试、回退和人工介入

### AI Coding 工具
- **代表**: Claude Code、Codex CLI、Cursor、Hermes Agent
- **价值**: 快速生成样板代码、解释陌生库、加速原型
- **限制**: 代码审查、测试设计、架构决策无法被替代

### AI 输出验证（5 大风险）
1. **事实错误**: 自信编造不存在的信息，关键事实须外部核实
2. **引用错误**: 编造论文、链接、数据来源，不信任模型给的链接
3. **推理漂移**: 长上下文中逻辑链断裂，结论偏离前提
4. **执行越权**: agent 做了超出授权范围的操作，须设置 guardrails
5. **工具误用**: 调用了错误的工具或参数，须用 tracing 监控

## Web3 方向

### 账户、地址和钱包
- **账户**: 链上身份，由私钥控制
- **地址**: 账户的公开标识（公钥的哈希）
- **钱包**: 管理私钥的工具，不是"存放"资产的地方
- **关键**: 私钥 = 资产控制权，泄露 = 资产丢失

### 助记词、私钥、地址
- **助记词**: 12/24 个单词，生成私钥的种子
- **私钥**: 256 位随机数，控制账户
- **地址**: 从私钥推导出的公开标识
- **安全底线**: 助记词和私钥永远不能泄露、输入、粘贴或让 agent 读取

### 签名与交易
- **签名**: 授权一个具体动作，不是"点一下确认"
- **交易**: 签名后的指令，广播到网络执行
- **关键**: 签名 = 授权，交易 = 执行

### Gas
- **是什么**: 链上执行的计算资源定价
- **为什么**: 防止滥用、激励矿工/验证者
- **特点**: 失败也会消耗 Gas、价格由市场决定

### 智能合约
- **是什么**: 部署在链上的程序，自动执行预定义逻辑
- **特点**: 状态公开、执行公开、升级权限可检查
- **与后端的区别**: 无法随意修改、执行成本高、透明度高

### 主网与测试网
- **主网**: 真实资产、真实成本
- **测试网**: 测试用、免费测试币、无真实价值
- **原则**: 学习和实验应先在测试网完成

### 区块浏览器
- **是什么**: 查看链上数据的工具（如 Etherscan）
- **用途**: 查看交易状态、合约代码、账户余额

## AI × Web3 交叉

### 为什么交叉是趋势
- AI 擅长理解意图、生成内容、做出决策
- Web3 擅长去信任执行、资产控制、透明记录
- 交叉点：AI Agent 自主执行链上操作、可验证 AI 推理、链上数据智能分析

### 关键基础设施
- **账户抽象 (ERC-4337)**: 让 AI Agent 可以管理智能账户
- **Session Key**: 限制 AI Agent 的权限范围
- **Guardrails**: 防止 AI Agent 越权操作
